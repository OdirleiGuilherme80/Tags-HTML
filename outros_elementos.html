Elementos do HTML

Elemento B e Strong(negrito e ênfase para leitores de navegação)

O elementos b e strong tem como função destacar uma palavra, frase ou texto em negrito. Mas sabemos que toda página web deve ser construida para todo tipo de usuário, principalmente aos deficientes visuais. Pensando nisso na hora de destacar qual palavra em negrito, utilizaremos o "strong" com ele o leitor de navegação e capaz de dar uma forte ênfase na palavra negritada. Já o elemento b somente irá deixar em negrito, mas o leitor de navegação não realizara tal leitura com dando ênfase a palavra.
Exemplos abaixo =>

  • O menino jogou a bola no mato!

  • A menina corajosa que entrou no mato e buscou a Bolaaa!

Elementos I e EM (itálico em ênfase)

Bom, a explicação para o itálico segue o mesmo príncipio do negrito, ambos tem a mesma função. Pórem utilizaremos o "em" para dar ênfase para os leitores de navegação.
Exemplos abaixo =>

  • Todo texto com um efeito Itálico fica um charme.

  • Construa uma frase com itálico: Quero uma pizza italiana
